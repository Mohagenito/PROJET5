# Projet5_Cat-gorisez automatiquement des questions

## Objectif du Projet
Cette API Flask utilise des modèles SBERT et Classifier Chain pour prédire des mots clés à partir de questions.

## Structure du Répertoire
- `app/` : Contient le code de l'API Flask.
- `tests/` : Contient les tests unitaires.
- `docs/` : Contient la documentation du projet, y compris le fichier introductif et la liste des packages utilisés.

## Installation
Pour installer les dépendances, utilisez :
```bash
pip install -r requirements.txt
